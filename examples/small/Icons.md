# Icons

Apply an icon to the theme HTML or markdown using the integrated Google Material Icons.

Apply the `<i>icon_name</i>` (`i` for _icon_) within your markup:

    <a href="#">
        <i>edit_note</i>
        Edit
    </a>

This will produce something similar to:

<a href="#">
    <i>edit_note</i>
    Edit
</a>
